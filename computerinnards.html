Why Should You Know What's In a Computer?

People who are more than casually interested in computers should have some idea what the underlying hardware is. Donald Knuth, Turing Award winner, Professor of Computer Science 		question_answer

You should know what is inside a computer. You will need to compare computers in order to decide which one is the best fit for you or your company.

You should know the component parts in a computer system. Understanding a computer spec is important in deciding what the base line is for your computer system, home or work, and judging whether to repair or replace.

Lossy compression is a method of data compression in which the size of the file is reduced by eliminating data in the file. Image quality is sacrificed to decrease file size.

Standard .jpg files represent lossy compression.

Lossless compression allows the original data to be fully reconstructed from the compressed data. Lossless compression methods are reversible.

Standard .png files represent lossless compression.

A repository is a storage space for project files. It can be local to a folder on your computer, or it can be a storage space on GitHub or another online host. You can keep code files, text files, images or any kind of a file in a repository.

URL is an acronym for Uniform Resource Locator and is a reference (an address) to a resource on the Internet.

A URL has two main components:
Protocol identifier: For the URL http://example.com, the protocol identifier is http.

This involves getting data and instructions into the computer, via special devices referred to as input devices.

The results of a computation are delivered to the user, via special devices referred to as output devices.

Primary storage is also known as RAM (Random Access Memory). This is the storage that is volatile, content disappears when power goes. Secondary storage is permanent storage, like on a disk.

The motherboard is the main circuit in a computer system.

Small wire like etchings, termed traces,convey data between the micro-components and parts of the board.

The CPU or Central Processing is the chip that is described as the brains of the computer.

Modern CPUs subsume functionality of an ALU (Arithmetic and Logic Unit), performing calculations.

An HDD, Hard Disk Drive, is a permanent storage device with spinning magnetic disks and a read/write head to encode and read the information on the disk.

A SSD (Solid State Device) is a newer permanent storage device that uses flash memory, like a USB drive. This memory can be electrically erased and reprogrammed, and contents persist even in the absence of power.

RAM, Random Access Memory, is primary storage- the storage that is used in processing data or running a computer program. RAM is volatile, which means that the content of RAM is only persisted while there is a power source.

Basic Input Output System, or BIOS is firmware (software that is pre-installed). A BIOS chip, is typically a core part of a motherboard and is used to perform hardware initialization during the booting process (when a computer is powered on).

Delicate computer component parts need direct current(DC), but house supply is alternating current(AC).

The Power Supply Unit (PSU) regulates the power supply and converts it to DC.

A graphics card, or video card, has its own memory and resources so it can improve video playback quality, which is especially important with high definition video. Additionally, a graphics card is designed to process and compress video and helps a computer process a video faster.

If on-board graphics functionality fails, a graphics card can be installed to restore processing capability.

A sound card reduces CPU usage, freeing it up for other tasks. The sound card takes on the audio load of the computer system, resulting in better sound quality and better performance.

If on-board sound functionality fails, a sound card can be installed to restore audio processing capability.

A NetworkInterface Card (NIC) can add or replace wired or wireless capability and impact network connection functionality in a computer system.

GitHub Pages is a web site hosting service that takes HTML, CSS, and JavaScript files straight from a repository on GitHub, and publishes a website.

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere that has internet access.

The clock speed of a processor is a measure of the number of instructions executed per second. Clock speed is measured in hertz, specifically today, in GigaHertz (billions of instructions per second).

A processor core is a unit that performs the processing. In the early days of computers, the core was in essence the processor. The evolution of the technology led to processors having multiple cores, to allow for tasks to be done seemingly at the same time.

Threads of execution describe the flow of execution for a particular task.

Sometimes, inside a processor core, one task is interrupted for a period of time, while another one executes. This looks like 2 tasks are running at the same time, but it is not the case.

Cache (from the French word for hidden) memory, is memory that typically resides inside the processor casing. This cache memory saves results, to improve the efficiency of execution.

There are different levels of cache memory, representing access speed and proximity to the CPU core(s).

An instruction set is code that the processor can understand. ISA, Instruction Set Architecture, can be composed of instructions of varying numbers of bits. These instructions control the flow of control and execution inside the processor core. The wider (in bits) the instruction set, the more instructions. More instructions is typically best, because less compositions of instructions are necessary.

A multi-core processor is a single processor unit, containing two or more separate processing units - cores. Each core a separate entity, reading and executing the Instruction Set commands.

Von Neumann architecture is the architecture upon which nearly all digital computers have been based. Characteristics include

  • centralized control, housed in the central processing unit
  • a separate storage area, primary memory, which can contain both instructions and data - instructions are executed by the CPU, and are brought into the CPU from primary memory
  • a unit that performs operations on operands, the arithmetic and logic unit (ALU)
